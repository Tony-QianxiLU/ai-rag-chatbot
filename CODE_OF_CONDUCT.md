# Code of Conduct

## Our Pledge

Contributors and maintainers should make this project a respectful, welcoming, and harassment-free environment.

## Expected Behavior

- Be respectful and constructive.
- Give feedback that helps improve the project.
- Assume good intent while staying accountable for impact.
- Keep technical discussions focused on the work.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without permission.
- Repeated disruption of project discussions.

## Enforcement

Maintainers may remove comments, close issues, or block participants when behavior is inconsistent with this code of conduct.
